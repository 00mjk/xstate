# @xstate/devtools

Developer tools for XState.

# Quick Start

  • Run npm install
  • Run npm start to build the /panel in development mode
  • Go to chrome://extensions and Load unpacked this directory (/packages/xstate-dev)
  • Select the XState extension in the dev tools
